# Image-Processing-project

## API for resize image

```
- /api/resizeImage?filename=map&width=400&height=400
```
## API route access cache

```
 /api/cache
```

### Start the server
```
  npm i
  npm start
```
### Testing using Jasmine

```
  npm run test
```

### to run Linting

```
  npm run lint
```

### to run Prettier

```
 npm run prettier

```
